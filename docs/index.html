textxover

Live scrolling comments, emoji reactions, and fireworks on your macOS screen. GPU-accelerated, visible in screen sharing.

Download for macOS View on GitHub
Menu bar controls
Anyone can comment via WebUI
Live polls with real-time results
Mini window (Picture-in-Picture)

Features

💬

Scrolling Comments

Niconico-style comments flowing across the screen. Scroll, top-fixed, or bottom-fixed.

🎆

Fireworks

GPU-computed particle fireworks with realistic physics, trails, and color transitions.

😍

Emoji Reactions

One-tap emoji reactions that appear on the overlay in real-time.

📊

Live Polls

Start polls from the host UI. Participants vote from their browser, results update live.

🔗

One-Click Sharing

Share a public URL via cloudflared tunnel. No setup needed — just click and share.

⚡

GPU-Accelerated

wgpu/Metal rendering at 60fps. Transparent overlay visible in screen sharing.

Quick Start

# Install just (if not installed)
brew install just

# Clone and run
git clone https://github.com/keisuke-na/textxover.git
cd textxover
just run
